
REQUIREMENTS
------------

- Drupal 7.x


INSTALLATION
------------

- Unpack the "checklist" directory into your "modules" directory

- Download and install the module dependencies (Usergroups API)

- Enable the module in administer > modules

- Database tables will be created automatically

- Grant the proper access to user roles under administer > access control

- Setup checklist(s) at admin/structure/checklist
